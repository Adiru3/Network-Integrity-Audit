# Gaming Network Integrity Audit

![Network Integrity](https://img.shields.io/badge/Performance-Esports--Ready-orange?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

A professional-grade network diagnostic tool designed for gamers and competitive players. Unlike standard speed tests, this audit simulates real-world gaming engine loads (128 Tick-rate) and analyzes packet stability at a granular level.

## üõ† Key Metrics
* **Bufferbloat Delta:** Measures latency spikes under network congestion.
* **Jitter (Std Dev):** Evaluates ping consistency‚Äîcrucial for games like CS2, Valorant, and Apex Legends.
* **MTU Sensitivity:** Analyzes how packet size impacts your overall delay.
* **Burst Loss Analysis:** Identifies patterns in packet loss to find hardware or ISP faults.
* **Predicted Lag:** Forecasts real-world input delay based on current jitter and RTT.

## üìä Features
* **Real-time Charting:** Visualize your connection stability in real-time.
* **Baseline Comparison:** Save your "ideal" results to compare against future lag spikes.

* **Pro Standards Table:** Compare your results against Esports industry benchmarks.
